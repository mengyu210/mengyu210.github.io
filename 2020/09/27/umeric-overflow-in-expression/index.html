发布于 2020-09-27   更新于 2020-09-27 Java int数值溢出

代码

代码校验提示 Numeric overflow in expression 错误表达式溢出

原因

在于java不具有‘目标确定类型’的特性（指存储结果的变量的类型会影响到计算所使用的变量类型）

赋值uploadSize赋值过程如下

  • 1、先用int 类型计算5 * 1024 * 1024 * 1024 ,并用int存储(如果值大于int,则截取int数据的部分)
  • 2、然后把int值转为long

总结

在long类型赋值计算时，不要忘记在变量后面加上L ，标注变量为Long类型
